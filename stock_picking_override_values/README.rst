===========================
Override new picking values
===========================

Allow overriding new picking values with system parameters.
The main purpose is to allow overriding the picking company,
but can be used for other fields too.

Add a system paremeter named "stock.override.<field_name>" to override.
The override can't be used to set value as empty or False.

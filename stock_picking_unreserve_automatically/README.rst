=======================================
Stock Picking - Unreserve automatically
=======================================

* Set sale orders to automatically unreserve deliveries

Configuration
=============
Go to Sale settings via Sales -> Configuration -> Settings
and select Force Unreserve to always unreserve deliveries
from sale orders.

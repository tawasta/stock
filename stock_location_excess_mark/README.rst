==================================
Mark stock location as Excess type
==================================

This module is used to mark stock location as of being excess type.
There is not a boolean field in standard odoo to be used for this.

Configuration
=============

Usage
=====
Select a stock location and find its "Is an Excess Location" field to mark it
as one.
